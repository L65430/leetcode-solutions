## 878. 第 N 个神奇数字  


题意：如果正整数可以被 A 或 B 整除，那么它是神奇的。
返回第 N 个神奇数字。由于答案可能非常大，返回它模 `10^9 + 7` 的结果。


思路：简单分析后，可以发现神奇数字是有规律的，即以最大公倍数 G 为一个周期，假设G内有 m 个神奇数。  
则需要求第N个神奇数时，这个神奇数肯定是 [G * (N/m) + f(N%m)内。  
`f(k)`的含义是第k个神奇数。  
由于 G 内的神奇数时有限的，所以很容易计算出来，从而计算出答案。
